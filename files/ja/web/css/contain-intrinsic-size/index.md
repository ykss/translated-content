{{CSSRef}}

`contains-intrinsic-size` CSS プロパティは、[`content-visibility`](/ja/docs/Web/CSS/content-visibility)で指定された要素の自然なサイズを制御します。

## Syntax

```css
/* <length> values */
contain-intrinsic-size: 1000px;
contain-intrinsic-size: 10rem;

/* <percentage> value */
contain-intrinsic-size: 10%;
```
